# dotfiles
Simple repo for my .dotfiles

## Set-up

### Shell

1. Install ZSH
2. Install oh-my-zsh
3. Install missing plugins
4. Symlink .zshrc

### VIM

1. Install VIM
1. Install Vundle
2. Install plugins `vi +PluginInstall +qall`

### TMUX

1. Install tmux
2. Install tpm
3. Install plugins `~/.tmux/plugins/tpm/tpm`

### Dev Tooling

1. Install rvm
2. Install nvm
3. Install tfenv
4. Install docker
5. Install pyenv
